# 📌 Project Overview

The objective was to perform **Exploratory Data Analysis (EDA)** on a cleaned sales dataset, uncover meaningful business insights, analyze sales trends, and build an **Executive Business Intelligence Dashboard** using Microsoft Excel.

---

# 🎯 Objectives

- 📈 Perform Exploratory Data Analysis (EDA)
- 🧹 Understand the cleaned sales dataset
- 📊 Identify sales trends and business insights
- 📉 Visualize data using charts
- 🗄️ Perform SQL-based business analysis
- 📋 Design an Executive Dashboard in Microsoft Excel

# 💡 Key Business Insights

- 🏆 Electronics generated the highest revenue.
- 💻 Laptops were the highest revenue-generating product.
- 🏙️ Patna recorded the highest sales.
- 👨 Male and 👩 Female customers contributed almost equally.
- 📈 Unit Price showed the strongest positive correlation with Total Sales.
- 📅 Sales remained relatively consistent throughout the year with noticeable peaks.
